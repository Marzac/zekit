Firmware Release Notes
======================
V2.0 - 03/01/2022
- Adding vibrato using the modwheel
- Adding a clock setting page (press wave + options)
- MIDI clock can now be enabled or disabled
- EXT clock can now be enabled or disabled
- Clock divider for EXT clock (from 1 to 4) 
- Faster navigation through UI pages
- Legato in mono mode (when holding a note)
- Glide time twice as long as before
- Fixing pitchbend role on chords release

V1.1 - 01/12/2021
- Fixing a rare envelope glitch (wouldn't retrigger)
- Enabling option to add ties to sequences
- Fixing saving feature of master tuning

V1.0 - 29/09/2021
Initial firmware release
